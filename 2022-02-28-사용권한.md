## 📌Read, Write, Execute 권한
**<폴더인지 파일인지 확인하기>**   

폴더와 파일을 생성하기 위해 명령어 mkdir 과 nano 를 이용한다.
```
mkdir linux
nano helloworld.js
```

>코드를 작성한 다음 Ctrl + X, Y, Enter 를 순서대로 입력하면, 새로운 파일을 저장할 수 있습니다.
