## 📌 MySQL 명령어 정리
**💡MySQL 실행**   
brew services start mysql

**💡MySQL 접속**   
mysql -u root -p
// -u(계정 접근), -p(비밀번호)

**💡DB 보기**   
show databases;

**💡DB 사용하기**   
use 데이타베이스명;

**💡DB 삭제하기**   
drop databse 데이타베이스명;

**💡DB 생성하기**   
CREATE DATABASE 데이터베이스_이름;

**💡테이블 보기**   
show tables;

**💡테이블 정보보기**   
DESCRIBE user;
